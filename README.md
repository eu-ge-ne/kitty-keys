## Installation

### Deno

```bash
deno add jsr:@eu-ge-ne/kitty-keys
```

### Node.js

```bash
# pnpm
pnpm i jsr:@eu-ge-ne/kitty-keys

# yarn
yarn add jsr:@eu-ge-ne/kitty-keys

# npm
npx jsr add @eu-ge-ne/kitty-keys
```

### Bun

```bash
bunx jsr add @eu-ge-ne/kitty-keys
```
